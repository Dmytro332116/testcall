### 3. НАЛАШТУЙ .env

**Створи файл `.env` в корені папки:**

```env
# ═══════════════════════════════════════════════════════════════
# SIP КОНФІГ (ТВІЙ СІП СЕРВЕР)
# ═══════════════════════════════════════════════════════════════
SIP_SERVER=62.149.25.79
SIP_PORT=5168
SIP_LOGIN=422
SIP_PASSWORD=YOUR_NEW_SECURE_PASSWORD  # ⚠️ НОВИЙ пароль (не T6ecK2zjH8xI!)

# ═══════════════════════════════════════════════════════════════
# ELEVENLABS API (СИНТЕЗ МОВИ)
# ═══════════════════════════════════════════════════════════════
# 🌐 Йди на: https://elevenlabs.io/app/settings/api-keys
# 📝 Копіюй ключ і вставь:
ELEVENLABS_API_KEY=sk_XXXXXXXXXXXXXXXXXXXXXXXXXXXXX

# ═══════════════════════════════════════════════════════════════
# OPENAI API (GPT-4 + WHISPER)
# ═══════════════════════════════════════════════════════════════
# 🌐 Йди на: https://platform.openai.com/api-keys
# 📝 Копіюй ключ і вставь:
OPENAI_API_KEY=sk-proj-XXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

# ═══════════════════════════════════════════════════════════════
# ЛОГУВАННЯ
# ═══════════════════════════════════════════════════════════════
LOG_LEVEL=debug
NODE_ENV=production
```

**⚠️ КРИТИЧНО ВАЖНО:**
- `SIP_PASSWORD` - використовуй НОВИЙ пароль, не старий!
- `ELEVENLABS_API_KEY` - регенеруй на сайті
- `OPENAI_API_KEY` - отримай від OpenAI

## 🔑 ОТРИМАННЯ КЛЮЧЕЙ

### 1. OpenAI API Key

```
1. Йди на https://platform.openai.com/api-keys
2. Натисни "Create new secret key"
3. Копіюй ключ (починається з sk-proj-)
4. Вставь в .env: OPENAI_API_KEY=sk-proj-...
```

**Ціна:** ~$0.01 за дзвінок (5 хвилин)

### 2. ElevenLabs API Key

```
1. Йди на https://elevenlabs.io/app/settings/api-keys
2. Якщо немаєш акаунту, зареєструйся (безкоштовно)
3. Натисни "Create new API key"
4. Копіюй ключ (починається з sk_)
5. Вставь в .env: ELEVENLABS_API_KEY=sk_...
```

**План:** Starter - безкоштовно (10k символів/місяць)

### 3. SIP Password

```
1. Змініть пароль у своєї хмари SIP сервера
2. Або попроси в технічної підтримки
3. Вставь в .env: SIP_PASSWORD=YOUR_PASSWORD
```

## ❓ ЧАСТЫЕ ВОПРОСЫ

### Q: Як далеко можу дзвонити?

**A:** На будь-який номер мобільної/стаціонарної телефонії через твій SIP провайдер. Межі залежать від акредитивів аккаунту.

### Q: Скільки коштує?

**A:**
- OpenAI: ~$0.01 за дзвінок
- ElevenLabs: ~$0.001 за дзвінок
- SIP: залежить від провайдера (~$10-50/місяць)
- **Всього:** ~$30-80 на місяць

### Q: Чи працює на Windows?

**A:** Так, якщо встановлено Node.js. Або через Docker (універсально).

### Q: Чи працює українська мова?

**A:** ✅ Повністю підтримується! Whisper розпізнає українську, GPT-4 говорить українською, ElevenLabs має українські голоси.

### Q: Чи можу добавити свої сценарії?

**A:** ✅ Так! Вся логіка в scenarios.js. Копіюй функцію, модифікуй systemPrompt.

### Q: Чи є обмеження на кількість дзвінків?

**A:** Залежить від сервера. Один сервер = 10-20 одночасних. Для 1000 дзвінків/день потрібно 5 серверів (Docker Swarm/Kubernetes).
