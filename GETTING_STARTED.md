```env

SIP_LOGIN=422
SIP_PASSWORD=YOUR_NEW_PASSWORD  # ⚠️ НЕ ВИКОРИСТОВУЙ СТАРИЙ (T6ecK2zjH8xI)!

# 2. ELEVENLABS
# 🌐 Йди: https://elevenlabs.io/app/settings/api-keys
# 📋 Копіюй новий ключ і вставь:
ELEVENLABS_API_KEY=sk_XXXXX...

# 3. OPENAI
# 🌐 Йди: https://platform.openai.com/api-keys
# 📋 Копіюй ключ і вставь:
OPENAI_API_KEY=sk-proj-XXXXX...
```

**ВАЖНО:** Старі паролі/ключи більше НЕ використовуй!

### ШАГ 3️⃣ : ВСТАНОВЛЕННЯ (5 хв)

```bash
# Перейти в папку
cd ai-voice-agent

# Встановити залежності (якщо не встановлено)
npm install

# Це встановить:
# - sip.js (для SIP дзвінків)
# - axios (для API запитів)
# - dotenv (для конфіг)
# - jssip (розширена SIP)
# - ws (WebSocket)
```

### ШАГ 4️⃣ : ЗАПУСК (2 хв)

```bash

npm run start:v2

# Або базова версія:
npm start

# Або через Docker:
npm run docker:compose
```
